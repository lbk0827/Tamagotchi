# v0.1 성능 점검 가이드

## 목표
- Idle 상태에서 UI 끊김이 없는지 확인
- 메모리 급증 여부 확인

## 준비
1. 앱 실행: `npm run dev`
2. 펫을 Idle 상태로 둔다(입력/버튼 클릭 최소화)

## 체크 포인트
- LED 패널 하단의 `FPS` 값이 지속적으로 표시되는지 확인
- 하단 `Heap` 값이 짧은 시간에 급격히 계속 증가하지 않는지 확인
- 하단 `CPU` / `App MB` 값이 비정상적으로 고정 급등하지 않는지 확인
- 30분 이상 방치 시 버튼 입력/창 드래그가 정상인지 확인
- `save-state`가 `Retry`로 고정되지 않는지 확인

## 권장 합격 기준 (초안)
- 30분 방치 후에도 입력 지연 체감이 없음
- FPS가 장시간 20 이하로 고정되지 않음
- Heap이 단조 증가만 하지 않고 안정 범위에서 변동
- CPU가 장시간 60% 이상으로 고정되지 않음 (Idle 기준)
