# v0.1 QA 테스트 플랜

## 1. 기본 실행
- [ ] `npm run dev`로 앱이 정상 실행된다.
- [ ] 알 모양 창이 표시된다.
- [ ] 창이 항상 위에 표시된다.

## 2. 윈도우/히트 테스트
- [ ] 알 내부에서는 클릭/입력이 정상 동작한다.
- [ ] 알 바깥 투명 영역 클릭 시 바탕화면 아이콘 클릭이 가능하다.
- [ ] 알 영역 드래그로 창 이동이 가능하다.
- [ ] `-` 버튼 클릭 시 창이 숨겨진다.
- [ ] Tray 메뉴에서 `Show AI Pet`로 다시 표시된다.

## 3. 상태 시스템
- [ ] Feed 클릭 시 Hunger 증가, Mood 소폭 증가
- [ ] Play 클릭 시 Mood 증가, Energy/Cleanliness 감소
- [ ] Clean 클릭 시 Cleanliness/Health 증가
- [ ] Sleep 클릭 시 Energy 증가
- [ ] 1분 경과 시 상태 감쇠가 발생한다.

## 4. 저장/복구
- [ ] 앱 종료 후 재실행 시 마지막 상태가 복구된다.
- [ ] 앱을 10분 이상 껐다 켜면 오프라인 시간 반영이 보인다.

## 5. AI 대화
- [ ] `.env`에 `OPENAI_API_KEY` 설정 시 AI 응답이 나온다.
- [ ] 키가 없거나 네트워크가 끊기면 폴백 응답이 나온다.
- [ ] 응답 중 Send 버튼이 비활성화된다.
- [ ] 펫 이름/성격 변경 후 응답 톤이 반영된다.

## 6. 안정화/성능
- [ ] 30분 이상 실행 시 UI 프리징이 없다.
- [ ] 상태 저장 실패 시 사용자 안내 문구가 보인다.
- [ ] `LOG_LEVEL=debug`에서 로그가 상세히 출력된다.

## 7. 수용 기준 최종 확인
- [ ] 알 모양 플로팅 창 안정 동작
- [ ] 4개 액션 상태 반영
- [ ] 시간 경과 로직 정상
- [ ] AI 실패 시 폴백 정상
- [ ] 저장/복구 정상
